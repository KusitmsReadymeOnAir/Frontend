#### 🧩 구현기능
- 카테고리 분류 및 검색
- 글 작성 (이미지 삽입, 에디터 기능)
- 글 수정 및 삭제
- 댓글 및 대댓글
- 내가 쓴 글, 내가 댓글 단 글 조회
- 로그인 및 로그아웃

#### ⚒ 사용 프레임워크
- 프론트엔드 : React, Typescript, styled-component
- 백엔드 :  Node.js, Mongodb, AWS S3
- 디자인 : Figma
- 협업 툴 : GitHub , Notion

#### 💻 프론트엔드 실행 방법
> cd Frontend/readyme-onair-front
> 
> npm start
